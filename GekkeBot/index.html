Setup Guide
Add the bot, link your list, and configure everything
1

Add GekkeBot to your server

Click the link below and select the server you want to add the bot to.

Add Bot to Server
2

Create a list and copy its ID

In this dashboard, create a new list or use an existing one. Copy the list ID by clicking the copy button on the list card.

3

Link the list to your server

Go to your Discord server and use the command /link {id you copied}. The bot will link the list to your server.

4

Set up the Manager role

Inside your list settings, go to Manager and enter a role ID. Users with this role can use ban, kick, mute, unmute, and unban commands. These users cannot target members with a higher or equal role. Users with Administrator permission can always use these commands. This step is optional.

5

Choose what to configure

Now you can set up the features you want to use.

Go to Tickets > General in your list settings and activate tickets.
Enter the Channel ID where you want the ticket panel to appear.
Optionally set a Category ID where ticket channels will be created, and a Support Role ID that gets added to tickets.
Creator Can Close lets ticket creators close their own tickets.
The Ticket Message is sent when a ticket opens. Use placeholders:
[creator] pings the ticket creator
[support] pings the support role (if set)
[id] shows the ticket ID
In Tickets > Permissions you can customize what the ticket creator and support team can do inside the ticket channel.
In Tickets > Embed you can enable a custom embed. We recommend using discohook.org to build your embed, then click JSON Data Editor and paste the JSON into the embed input.
Add Reasons so users can select why they are creating a ticket from a dropdown menu.
In Tickets > Ticket Logs you can view all open and closed tickets, read the full message history including edited and deleted messages, and download any ticket as a JSON file.
Go to AI Moderator in your list settings and activate it.
Add your Rules that the AI will enforce. Each rule can have an optional timeout action set in seconds, minutes, hours, or days.
Optionally add Excluded Roles so users with those roles bypass the AI moderator.
Optionally add Excluded Channels so messages in those channels are not checked.
Silent Mode makes the bot delete messages and take actions without sending a public violation message in the channel.
